# MicroStream_Application
This repository deals with the code base of MicroStream application which is developed using Spring Boot, event-driven microservices, and Apache Kafka. In this application, the user can place an order with the help of REST API. The order details can be produced using Kafka producer and various microservices(modules) can consume the details which are generated by the producer, and those order details are then saved into the database and an email will be sent to that particular user.
